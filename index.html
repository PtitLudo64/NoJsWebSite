Lorsque nous construisons un site Web ces jours-ci, il y a 110% de chances qu'il y ait une forme de JavaScript dessus. Qu'il s'agisse d'un framework complet, pour les animations, pour déclencher une popup ou comme un script de suivi, JavaScript est tout autour de nous.

Mais que se passe-t-il si je vous disais que vous n'avez pas du tout besoin d'utiliser JavaScript? Pas même comme processus de construction? Grâce aux mises à jour des technologies de navigation, il existe maintenant une pléthore de fonctionnalités de navigateur natives qui permettent de créer des sites Web modernes et fonctionnels, sans JavaScript.

Ensemble, nous allons construire un site web complètement statique, une collection de fichiers HTML et CSS, pas de suivi, pas de script, pas de serveurs, pas de ressources tierces. Construisons un site Web comme nous le faisions auparavant (mais pas au marteau et burin).
